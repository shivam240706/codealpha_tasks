# 🚀 CodeAlpha Data Science Internship Portfolio

Welcome to my Data Science Internship repository! I'm Shivam, and this repository contains the complete, structured code and analysis from my internship with CodeAlpha. 

## 📌 Objective
The primary goal of this internship was to build a robust foundation in data manipulation, exploratory data analysis (EDA), predictive modeling, and translating mathematical machine learning outputs into actionable business strategies.

## 🛠️ Technology Stack
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Machine Learning:** Scikit-Learn (Regression Modeling, Evaluation Metrics)
* **Data Visualization:** Seaborn, Matplotlib
* **Environment:** Jupyter Notebook
